# scala-cli-sbt-pascal-triangle

## Description
Find binomial coeffient of Nth.

## Tech stack
- openjdk 8
- scala
- sbt

## Docker stack
- docker-cli
- openjdk:8-jre-alpine

## Requirements
Docker desktop must be installed and the application
being called must be linux compatiple.

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## To see help
`sudo ./install.sh -h`
